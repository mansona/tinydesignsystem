About Tiny Design System

This site is designed to challenge your ideas of what a "Design System" is. You don't need fancy colours, fonts, components, animations, JavaScript, or anything that makes up the modern web.

The only thing you need for a design system is a consistent set of rules, and bonus points if those rules are simple and easy to follow so that your applicaiton design is consistent.

This website is built using no build system, it is literally a GitHub repo that contains 2 HTML files that are deployed on Netlify. If you want a good tool to make a modern Design System that is not too complex to use then check out Field Guide and my talk on building a universal design system (coming soon).
